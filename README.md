# Simulateur financier BCV38

Page statique pour simuler le bilan financier du club de badminton en ajustant les licenciés, cotisations, volants, entraînements, stages, manifestations et avantages club.

## Ouvrir localement

Ouvrir `index.html` dans un navigateur.

## Publication

Le site est compatible avec GitHub Pages sans étape de build.
